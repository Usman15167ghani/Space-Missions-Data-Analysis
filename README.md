# Space Missions Data Analysis & Visualization
## Project Overview

This project presents an exploratory data analysis of global space missions. The project analyzes historical space launch data to identify trends, mission success rates, leading launch companies, rocket usage, geographical patterns, and rocket costs.
The main goal of this project is to apply data cleaning, exploratory data analysis, data visualization, and statistical analysis techniques using Python.

## Objectives
- Analyze global space mission trends over time.
- Identify companies with the highest number of space launches.
- Analyze mission success and failure rates.
- Identify the most frequently used rockets.
- Explore geographical patterns in global space launches.
- Analyze rocket costs and detect cost outliers.
- Investigate relationships between mission success and other variables.

## Statistical Analysis
Chi-square hypothesis testing was used to investigate differences in mission success rates across launch companies and historical periods.
Correlation analysis was performed to examine the relationship between rocket cost and mission success.

## Key Insights
- Global space launch activity is concentrated among a limited number of major launch companies.
- Mission success rates are generally high, although launch failures still occur.
- Space launch activity changes over time and shows historical growth patterns.
- A small number of rocket platforms are used frequently for space missions.
- Global space launch activity is geographically concentrated.
- Rocket cost alone has a weak relationship with mission success.
